ESP32 Version Info Test
=======================

This simple app tests if can get the version information from the ESP32 chip.
If so, this indicates that the kernel is setup correctly and communication
with the ESP32 is working as expected.

Expected Output
---------------

```
Version Info:
AT version:1.3.0.0(Oct 28 2016 11:29:39)
SDK version:2.0.0(6ccde98)
compile time:Mar 31 2017 14:50:03
```
